### Summary

### setup
 
* PUBLIC_TRPC_URL: tPRC server
* auth info: PUBLIC_BASIC_AUTH_USER, PUBLIC_BASIC_AUTH_PASSWORD

```
PUBLIC_BASIC_AUTH_USER = "hoge"
PUBLIC_BASIC_AUTH_PASSWORD = "1111"

PUBLIC_TRPC_URL=http://localhost:4000/trpc
```

npm i
